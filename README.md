# Laravel React Template

Template generico para acelerar o inicio de novos projetos com Laravel + React.

## Visao Geral

- Backend em `Laravel 12`
- Frontend em `React 19 + TypeScript + Vite`
- UI com `MUI`
- Graficos com `Recharts`
- Banco de dados MySQL

## Estrutura

- `app/`: regras de negocio, controllers, models e services
- `resources/ts/`: frontend React/TypeScript
- `routes/`: rotas web e API
- `database/`: migrations e seeders
- `docker/`: configuracao do Nginx e scripts de banco
- `docker-compose.yml`: servicos auxiliares (MySQL local)

## Requisitos

- PHP `^8.2`
- Composer
- Node.js e npm
- MySQL
