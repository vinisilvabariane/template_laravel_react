# Arquitetura

## Visao geral

Este template e uma base Laravel com frontend React servido por Inertia.js.

Principais tecnologias:

- Backend: Laravel 12 com PHP 8.2
- Frontend: React 19 com TypeScript
- Integracao frontend/backend: Inertia.js
- Build frontend: Vite
- Interface: MUI
- Banco de dados: MySQL externo ao ambiente Docker
